# Netflix Content Analysis Dashboard (Tableau)

**Summary:**
This project features an Exploratory data analysis of Netflix’s content catalog (~8,800 titles) to identify trends in content growth, genre composition, geographic distribution, and audience targeting. The project demonstrates data cleaning, aggregation, and dashboard-based storytelling using Tableau.

## Key Insights
- Netflix’s content library expanded rapidly between 2016–2019, driven primarily by movie releases.
- Drama and Comedy dominate the catalog, indicating prioritization of broad-appeal genres.
- The U.S. remains the primary content source, with strong growth from India and the U.K., reflecting global expansion.
- Content ratings skew toward TV-MA and TV-14, suggesting a focus on mature and teen audiences.
